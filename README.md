# Disease Trend Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard that analyzes disease trends across multiple districts. The goal of this project is to transform raw health data into meaningful insights to help monitor disease outbreaks and support data-driven decision making.

The dashboard visualizes confirmed and suspected disease cases and allows users to explore trends by disease type, month, and location.

---

## Dashboard Features

• Confirmed vs Suspected case comparison  
• Disease-wise case distribution  
• Monthly disease trend analysis  
• District-level disease monitoring  
• Top 5 affected districts visualization  
• Interactive filters for Month, Disease, and Location  
• Drill-through pages for detailed disease analysis  

---

## Diseases Analyzed

- Dengue
- Influenza
- Malaria
- Typhoid
- Leptospirosis
- Chickenpox
- Viral Hepatitis A (VHA)

---

## Tools & Technologies

- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX
- Data Visualization
- Dashboard Design

---

## Key Insights

This dashboard helps identify:

- Diseases with the highest confirmed cases
- Monthly outbreak trends
- Districts most affected by specific diseases
- Differences between suspected and confirmed cases
